## 🎨 프로젝트 명: "Solve.it"

**컨셉:** Apple 스타일의 미니멀리즘 + Toss의 사용성 (깔끔하고 직관적인 UI)

### 1. 기술 스택 및 환경

* **Frontend:** HTML5, Tailwind CSS (디자인), Vanilla JS (가벼운 로딩)
* **Deployment:** GitHub Pages (Static Hosting)
* **Design System:** * **Font:** Pretendard (가독성 높은 산세리프)
* **Color:** `#FFFFFF`(배경), `#007AFF`(포인트 블루), `#F2F2F7`(카드 배경)
* **Style:** Rounded corners (16px+), Soft shadows, Glassmorphism 효과



### 2. 핵심 웹페이지 구성 요소 (Claude 요청용)

#### **A. 헤더 (Sticky Header)**

* 좌측에 깔끔한 텍스트 로고.
* 우측에 '다크모드 토글' 버튼과 '서비스 제안' 버튼.
* 배경은 반투명한 블러 처리(Backdrop Blur).

#### **B. 히어로 섹션 (Hero Section)**

* **카피:** "당신의 일상을 가볍게 만드는 모든 도구" (세련된 타이포그래피)
* **검색창:** 중앙에 큰 Search Bar 배치 (예: '전역일' 검색 시 관련 카드로 바로 이동).

#### **C. 카드형 서비스 리스트 (Main Grid)**

* 각 도구(계산기 등)를 독립된 **'카드'** 형태로 배치.
* 카드 구성: 아이콘(Lucide-react 스타일), 제목, 짧은 설명, '바로가기' 버튼.
* **Hover Effect:** 마우스를 올리면 카드가 살짝 위로 올라가는 부드러운 애니메이션.

#### **D. 푸터 (Footer) & SEO 영역**

* 사이트 맵과 저작권 표시.
* **광고 슬롯:** 구글 애드센스가 들어갈 하단 고정 배너 영역 확보.

### 3. Claude Code를 위한 정교한 프롬프트 (Copy & Paste)

> **"GitHub Pages용 모던 유틸리티 포털 사이트를 제작해줘."**
> **[요구사항]**
> 1. **Framework:** 프레임워크 없이 HTML과 Tailwind CSS CDN만 사용해서 단일 파일(index.html) 혹은 간단한 구조로 만들어줘.
> 2. **Design:** >    - Apple 공식 홈페이지 같은 미니멀하고 세련된 화이트톤 UI.
> * `Pretendard` 폰트를 적용하고, 카드에는 `rounded-2xl`과 부드러운 `shadow-sm`을 사용해줘.
> * 반응형 디자인 필수 (모바일에서 1열, PC에서 3열 배치).
> 
> 
> 3. **Components:**
> * 상단 네비게이션바 (로고, 검색 아이콘).
> * 메인 히어로 섹션 (문구: "심플하지만 강력한 도구 모음").
> * 서비스 카드 3개 (샘플 데이터: '군 전역일 계산기', '주식 수익률 계산기', '글자수 세기').
> 
> 
> 4. **Interaction:** >    - 각 카드에 마우스를 올리면 부드럽게 커지는 효과 추가.
> * 검색창에 텍스트 입력 시 카드가 필터링되는 간단한 JS 로직 포함.
> 
> 
> 5. **AdSpace:** 상단 헤더 아래와 하단 푸터 위에 '광고 영역'을 명확히 디자인적 요소로 포함해줘.
